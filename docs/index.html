DataImport

I upload a data dictionary located at {string} to project ID {int}

Parameters
data_dictionary (string) the path to the desired data dictionary located within the /fixtures/dictionaries/ folder.

Uploads a data dictionary to a specific project given a Project ID.

Imports well-formed REDCap data import file to a specific project given a Project ID.
